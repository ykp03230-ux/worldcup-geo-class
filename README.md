# 월드컵 세계지리 앱 GitHub 단일파일판

이 버전은 유니폼 이미지를 index.html 안에 직접 포함한 GitHub Pages용 안전 버전입니다.

업로드할 파일:
- index.html
- records.html
- quiz.html
- README.md

assets 폴더가 없어도 메인 지도 화면의 유니폼 이미지가 표시됩니다.


## v32 수정 사항
- 모바일 지도에서 두 손가락 핀치 줌(확대·축소)을 지원하도록 수정했습니다.
- 핀치 줌 중 두 손가락을 움직이면 지도 이동도 함께 가능합니다.
- 기존 드래그 이동, + / - 버튼, 마우스 휠 확대 기능은 유지했습니다.
