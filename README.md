## 변경 사항
- 48개국의 '월드컵 이모저모' 문장을 전면 수정했습니다.
- 단순 설명체를 줄이고, 학생들이 흥미를 느끼도록 '알고 있었나요?' 느낌의 이야기형 문장으로 바꿨습니다.
- 에콰도르는 나라 이름이 스페인어 '적도'와 연결된다는 설명을 자세히 넣었습니다.
- 호날두, 크레이그 고든, 길베르토 모라, 첫 출전국, 선수단 추정 시장가치, 오래된 복귀국 등 흥미 요소를 반영했습니다.
