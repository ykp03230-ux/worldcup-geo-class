# 월드컵 세계지리 앱 GitHub 단일파일판

이 버전은 유니폼 이미지를 index.html 안에 직접 포함한 GitHub Pages용 안전 버전입니다.

업로드할 파일:
- index.html
- records.html
- quiz.html
- README.md

assets 폴더가 없어도 메인 지도 화면의 유니폼 이미지가 표시됩니다.


## v32 수정 사항
- 모바일 지도에서 두 손가락 핀치 줌(확대·축소)을 지원하도록 수정했습니다.
- 핀치 줌 중 두 손가락을 움직이면 지도 이동도 함께 가능합니다.
- 기존 드래그 이동, + / - 버튼, 마우스 휠 확대 기능은 유지했습니다.


## v33 수정 사항
- 메인 지도 화면의 국가 정보에서 `대표 위도`, `대표 경도` 대신 `위도 범위(약)`, `경도 범위(약)`을 표시하도록 수정했습니다.
- 수도 위치는 `수도 좌표` 항목으로 따로 유지했습니다.
- 위도·경도 범위는 지도 윤곽 데이터를 기준으로 계산한 수업용 약값입니다.
- 저해상도 지도에 독립 윤곽이 없는 작은 섬 지역은 수도 기준 약값으로 표시됩니다.


## v34 수정 사항
- 세계지도 위에 위도선(북위 60°, 북위 30°, 적도 0°, 남위 30°, 남위 60°)을 표시했습니다.
- 세계지도 위에 본초자오선(경도 0°)을 표시했습니다.
- 지도 범례와 설명 문구에 본초자오선·위도선 안내를 추가했습니다.


## v35 수정 사항
- 세계지도 위에 경도선(서경 120°, 서경 60°, 동경 60°, 동경 120°)을 추가했습니다.
- 적도 0°, 본초자오선 0°, 주요 위도선 라벨을 함께 표시했습니다.
- 지도 범례에 위도선과 경도선 항목을 추가했습니다.
- 국가 상세 정보 카드에서 `수도 좌표` 항목을 제거했습니다.


## v36 수정 사항
- 나라 정보 카드의 순서를 `대륙 → 수도 → 월드컵 조 → FIFA 랭킹 → 위도 범위 → 경도 범위` 순서로 바꿨습니다.
- 본초자오선과 경도선이 육지 레이어 뒤에 가려지던 문제를 해결했습니다.
- 위도선, 경도선, 적도, 본초자오선이 지도 위에 더 선명하게 보이도록 선 굵기와 색을 조정했습니다.


## v37 수정 사항
- 경도선을 단순히 진하게 표시하는 방식이 아니라, `서경 120°`, `서경 60°`, `동경 60°`, `동경 120°` 숫자 라벨이 보이도록 수정했습니다.
- 본초자오선을 다른 색으로 구분하고, 지도 상단에 `본초자오선`, `경도 0°` 라벨을 표시했습니다.
- 좌표선은 너무 강하지 않게 다시 조정하고, 라벨 중심으로 읽히도록 수정했습니다.


## v38 수정 사항
- 지도 하단 안내 문구(메시지)를 숨김 처리했습니다.
- 본초자오선을 적도처럼 별도 색상(보라색)으로 더 분명하게 표시했습니다.
- 지도 하단에 경선별 경도 안내(서경 120°, 서경 60°, 경도 0°, 동경 60°, 동경 120°)를 숫자로 표시했습니다.
